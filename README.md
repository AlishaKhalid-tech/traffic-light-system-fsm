# Traffic Light System using Finite State Machine (FSM)
A **Traffic Light Control System** implemented in **C++** using the principles of a **Finite State Machine (FSM)**. This project was developed as part of the **Theory of Automata** course to demonstrate how finite state machines can model and control real-world sequential systems.

## Project Overview
Traffic signal controllers operate by transitioning through a predefined sequence of states. This project models those transitions using a **Finite State Machine (FSM)**, where each traffic light state changes to the next according to defined transition rules.The implementation provides a practical demonstration of how theoretical automata concepts can be applied to solve real-world control problems.

## Objectives
* Understand the fundamentals of **Finite State Machines (FSMs)**.
* Apply automata theory to a real-world traffic control system.
* Simulate state transitions between traffic signal phases.
* Demonstrate the practical implementation of FSM concepts using C++.

## State Transition Flow
RED
  │
  ▼
GREEN
  │
  ▼
YELLOW
  │
  ▼
RED
The traffic light continuously cycles through these states, following the transition rules defined by the finite state machine.
